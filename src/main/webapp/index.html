Installation

Jappy Framework is on Maven Central. Best way is to use gradle.

repositories {
 mavenCentral()
}

dependencies {
  compile 'com.jappyframework:jappy-framework:1.3'
}

Logging

Undoubtebly the most critical component of any application. Here are some examples on how to configure logging using Jappy Framework. 

/**
 * Default logging.
 */
Log LOG = Log.get("user");

/**
 * Console log all debug messages, and daily rotating files in /mnt/logs
 */
Log LOG = Log.get("user").console(Level.DEBUG).daily("/mnt/logs", Level.INFO);

/**
 * Email crashes to support and log everything to S3, creating a maximum 50MB file.
 */
Log LOG = Log.get("user").email("support@support.com", Level.ERROR).s3(Level.INFO, "test-bucket", "logs", "50m");

/**
 * You can also do asynchronous logging and use chaining to get any combination you want.
 */
Log LOG = Log.get("user").async().email("support@support.com", Level.ERROR).email("help@support.com", Level.WARN).console(Level.DEBUG);

/**
 * Once initialized, use it as a normal Log4J logger.
 */
LOG.info("A little info");
LOG.error("things gone wrong", e);
